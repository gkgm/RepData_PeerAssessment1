Are there differences in activity patterns between weekdays and weekends?

  1. Add new factor variable dayType with 2 levels - “weekday” and “weekend”

dayType <- function(dates) { f <- function(date) { if (weekdays(date) %in% c(“Saturday”, “Sunday”)) { “weekend” } else { “weekday” } } sapply(dates, f) }

activity$dayType <- as.factor(dayType(activity$date))

str(activity)

  1. Panel plot containing a time series plot of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all weekdays and weekends

library(lattice)

steps.interval <- aggregate(steps ~ interval + dayType, activity, mean)

xyplot(steps ~ interval | dayType, data=steps.interval, layout=c(2,1), type='l')
